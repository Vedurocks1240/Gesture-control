What's Happening?

This innovative project combines AI-powered gesture recognition with wireless communication to create a completely contactless control system for a robotic car.

Using just your hand movements captured by a computer camera, the system translates gestures into precise vehicle controls through a Quarky microcontroller and Bluetooth technology.

System workflow from gesture to movement

Tech Stack

Computer Vision

OpenCV-based hand tracking with mediapipe for real-time gesture recognition at 30fps.

WiFi

Low-latency wireless communication with 20ms response time between devices.

Quarky Microcontroller

Processes input signals and controls motor drivers with precision movement algorithms.

DC Gear Motors

High-torque motors with encoder feedback for accurate speed and position control.

How It Works

1. Gesture Detection

The webcam captures hand movements which are processed by our custom AI model to identify specific control gestures.

2. Data Transmission

Recognized gestures are converted to control commands and sent via Internet to the car's microcontroller.

3. Car Movement

The Quarky board interprets commands and controls the motors to execute precise movements in real-time.
